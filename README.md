# 🚀 JC Investments

## 📌 Descripción

Sistema web desarrollado con PHP y MySQL orientado a la gestión y administración de información.

Este proyecto permite realizar operaciones CRUD para administrar registros de manera organizada y eficiente.

---

## ✨ Funcionalidades

- Registro de usuarios
- Edición de usuarios
- Eliminación de usuarios
- Listado de usuarios
- Gestión de préstamos
- Validación de información
- Conexión a base de datos

---

## 🛠 Tecnologías utilizadas

- PHP
- HTML
- CSS
- JavaScript
- MySQL
- XAMPP

---

## ⚙ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/CarlosLizarazodev/jc-investments.git
```

2. Copiar la carpeta al directorio:

```bash
xampp/htdocs/
```

3. Importar la base de datos en phpMyAdmin

4. Iniciar Apache y MySQL

5. Abrir en el navegador:

```bash
http://localhost/jc_investments
```
